# Budget Tracker

A command-line budget tracking app built in Python.

## Features
- Add income and expense transactions
- View spending by category with visual bar chart
- Filter transactions by month
- Data persists via CSV storage
- Full test suite with pytest

## Tech Stack
- Python 3.12
- CSV file I/O
- pytest unit testing

## How to Run
python3 budget_tracker.py

## How to Run Tests
pytest test_budget_tracker.py -v
